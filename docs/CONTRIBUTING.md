# دليل المساهمة

مرحبًا! إذا كنت مهتمًا بالمساهمة في هذا المشروع، إليك بعض الإرشادات لتبدأ:

## كيفية البدء

1. قم بعمل Fork للمستودع.
2. أنشئ فرعًا ج��يدًا لمتطلباتك (على سبيل المثال: `feature/اسم-الميزة`).
3. أضف التغييرات الخاصة بك.
4. قم بعمل Commit للتغييرات مع رسالة واضحة تشرح ما قمت به.
5. افتح Pull Request.

## شروط المساهمة

- تأكد من أن الكود يتبع معايير كتابة الكود المستخدمة في المشروع.
- اجعل التعديلات صغيرة وقابلة للإدارة.
- يجب أن تتضمن اختبارات جديدة أو تتضمن تحديثات للاختبارات إذا قمت بتغيير الوظيفة.

## التواصل

إذا كان لديك أي استفسارات، لا تتردد في فتح قضية أو مراسلتنا مباشرة.
