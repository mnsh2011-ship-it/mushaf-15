# مشروع المصحف 15

هذا مشروع خاص بتطوير المصحف. تحتوي المستندات في هذا المجلد على معلومات حول كيفية استخدام المشروع والمساهمات. 

## المساهمة في المشروع

إذا كنت مهتمًا بالمساهمة في المشروع، يمكنك اتباع الإرشادات التالية:
1. قم بفورك هذا المستودع
2. أنشئ فرعًا جديدًا: `git checkout -b feature/اسم-المميزات`
3. قم بالتغييرات اللازمة
4. ادفع التغييرات: `git push origin feature/اسم-المميزات`
5. إنشاء طلب سحب (Pull Request)

## الاتصال

يمكنك الاتصال بفريق التطوير عبر البريد الإلكتروني أو فتح قضية (issue) في المستودع.
